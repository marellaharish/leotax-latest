# LEOTAXFILING (Vite + Tailwind + shadcn/ui style + Aceternity UI)

## Run
```bash
npm install
npm run dev
```

## shadcn/ui (Vite)
This project includes a shadcn-style setup (cn util + ui components).  
You can also use the shadcn CLI to add more components.

## Aceternity UI
Aceternity UI provides "magic effects" components and also supports installation via shadcn registry.

Example:
```bash
npx shadcn@latest add https://ui.aceternity.com/registry/bento-grid.json
```

Then import from `src/components` depending on where the CLI places it.
